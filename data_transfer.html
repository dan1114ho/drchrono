Data Format
Patient Data Format
Demographics .csv, .xls. CCDA
Appointments .csv (This needs to have the patient's name, the provider name (if multiple providers), the office/exam room, and the date/time of each appointment.)
Clinical Notes .pdf, .html (This needs to have the patient's name, the provider name (if multiple providers), the office/exam room, and the date/time of each appointment.)
Documents .pdf (The .pdf filenames need to have the patient's name, DOB, and optionally date associated with it.)
Images .pdf, .jpeg, .tiff (The filename needs to have the patient's name, DOB, and optionally date associated with it.)
Lab Results .pdf (The file name needs to have the patient's name, DOB, and optionally date associated with it.)
